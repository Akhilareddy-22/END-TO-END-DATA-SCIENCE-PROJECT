# DATA SCIENCE PROJECT: House Price Prediction Project

This project predicts house prices using Machine Learning and Flask.

## Project Structure
- data/ → dataset
- notebooks/ → model training notebook
- model/ → saved ML model
- templates/ → HTML frontend
- static/ → CSS styling
- app.py → Flask backend

## How to Run

1. Install libraries:
pip install -r requirements.txt 
Backend running
cd backend 
python app.py

2. Open browser:
http://127.0.0.1:5000/

## Input Features
- Area
- Bedrooms
- Bathrooms

## Output
Predicted house price.
